# What is a Cron Job? 
```sh 
* * * * * command(s)
^ ^ ^ ^ ^
| | | | |     
| | | | |     
| | | | ----- Day of week (0 - 7) (Sunday=0 or 7)
| | | ------- Month (1 - 12)
| | --------- Day of month (1 - 31)
| ----------- Hour (0 - 23)
------------- Minute (0 - 59)
```
